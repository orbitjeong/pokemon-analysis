# 🎮 **포켓몬 데이터 분석 (Pandas EDA 프로젝트)**

이 프로젝트는 **Pandas**를 활용해 포켓몬 데이터셋을 탐색(EDA)한 결과를 담고 있습니다.
공격력(Attack), 속도(Speed), 타입(Type 1) 분포, 그리고 물고기형(Fish-like) 포켓몬의 특징을 분석했습니다.

---

## 📁 **파일 구성**

* **pokemon_analysis.ipynb** — 전체 분석 과정이 담긴 Jupyter Notebook
* **pokemon.csv** — 원본 데이터셋

---

## 📌 **다룬 내용 (Topics Covered)**

* Pandas DataFrame 인덱싱 (`loc`, `iloc`)
* 데이터 정렬 및 필터링 (`sort_values`, `sort_index`)
* `value_counts()` 및 기초 통계 분석
* Pandas `.plot()`을 활용한 기본 시각화

---

## 📊 **핵심 인사이트 (Key Findings)**

* 공격력 상위 포켓몬은 **Dragon / Fighting 타입**이 다수
* 속도(Speed) 상위권은 **Electric / Flying 타입**이 높은 비중
* 물고기형(Fish-like) 포켓몬 중 **Gyarados의 공격력이 압도적**
* Pandas 기본 기능만으로도 충분히 의미 있는 EDA 수행 가능함을 확인
