# Pokémon Data Analysis (Pandas Beginner Project)

This project explores the Pokémon dataset using Pandas.
I analyzed Attack, Speed, Type distribution, and characteristics of fish-like Pokémon.

## Files
- `pokemon_analysis.ipynb`: Full analysis notebook
- `pokemon.csv`: Original dataset

## Topics Covered
- Pandas indexing (loc/iloc)
- Sorting and filtering data
- Value counts and descriptive statistics
- Data visualization with Pandas .plot()
